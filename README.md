# GitHub User Finder

Автор: Закирова Айзиля

## Описание

GitHub User Finder — GUI-приложение для поиска пользователей GitHub через GitHub API. Приложение показывает результаты поиска списком, позволяет открыть профиль пользователя в браузере, добавить пользователя в избранное и сохранить избранных пользователей в файл `favorites.json`.

## Требования

- Python 3.10 или новее
- Интернет-соединение для запросов к GitHub API

Дополнительные библиотеки устанавливать не нужно: приложение использует стандартную библиотеку Python (`tkinter`, `urllib`, `json`).

## Запуск

```bash
python main.py
```

## Как использовать API

Приложение обращается к GitHub Search API:

```text
GET https://api.github.com/search/users?q=<поисковый_запрос>&per_page=20
```

В запрос добавляются заголовки:

```text
Accept: application/vnd.github+json
User-Agent: GitHub-User-Finder-Tkinter
```

Для обычной работы токен не обязателен. Если GitHub ограничит количество запросов, можно создать персональный токен GitHub и перед запуском указать переменную окружения `GITHUB_TOKEN`.

Пример для PowerShell:

```powershell
$env:GITHUB_TOKEN="ваш_токен"
python main.py
```

## Примеры использования

1. Введите `torvalds` в поле поиска и нажмите `Найти`.
2. Выберите пользователя из списка результатов.
3. Нажмите `Добавить в избранное`.
4. Проверьте, что пользователь появился в списке избранных и записался в `favorites.json`.
5. Нажмите `Открыть профиль`, чтобы открыть страницу пользователя в браузере.

## Проверка ввода

Если поле поиска пустое, приложение показывает предупреждение и не отправляет запрос к API.
